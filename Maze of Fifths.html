Keep Talking and Nobody Explodes Mod Maze of Fifths

On the Subject of Maze of Fifths

Huge brain move: write a pop song while playing KTaNE.

Calculate the Manhattan distance* between the green and red cells of the maze. Add AA batteries. The last digit of the result is the number of the maze.

Build a path from green to white cell:

  • To start, press the green cell;
  • You can only move one cell orthogonally or diagonally per step;
  • If there is a bold border between orthogonal cells on the maze map, then this movement is prohibited;
  • If there is a circle between the diagonal cells on the maze map, then this movement is prohibited;
  • You can not visit the same cell twice;
  • You can not visit the red cell;
  • It is necessary to make exactly as many steps as the number of digits in the serial number multiplied by two (pressing the green cell at the start does not count as a step).

Maze #0

Designations of cell colors:

  • Green/Cyan: Start
  • Red: Forbidden
  • Blue/Cyan: Current position
  • White: Finish
  • Grey: Visited cell

The maze can be rotated and/or mirrored. Use Maze #0 and the letters on the module to find these transformations.

* The Manhattan distance is the smallest number of orthogonal steps needed to move between the two cells.
